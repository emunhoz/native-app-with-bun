<h1 align="center">React Native app with GraphQL (Apollo), Typescript, Expo and Bun!</h1>

## ⚠️ Requirements

- Bun 1.0.0
- Expo CLI

## 🚀 Quick start

Install all dependecies with:

```bash
  bun install
```

Run ios version 🍎

```bash
  bun run ios
```

Run ios version 🤖

```bash
  bun run android
```
